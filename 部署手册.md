# 部署手册（Gitee + 1Panel + Docker）

目标：**本地改代码 → push 到 Gitee → 服务器一行更新**，彻底告别手动传文件。

整套只需配置一次，之后每次更新上线 30 秒。

---

## 第一步：代码推到 Gitee（一次性）

### 1. 注册并建仓库
- 去 https://gitee.com 注册 / 登录。
- 右上角「+」→「新建仓库」：名称填 `wxwork_analytics`，选**私有**，**不要**勾选自动生成 README。
- 建好后复制仓库地址，形如 `https://gitee.com/你的用户名/wxwork_analytics.git`

### 2. 本地推送（在项目根目录的命令行里执行）
```powershell
cd C:\Users\Administrator\Desktop\wxwork_analytics
git add .
git commit -m "初始化：企微群会话分析系统"
git remote add origin https://gitee.com/你的用户名/wxwork_analytics.git
git push -u origin main
```
> 第一次 push 会让你登录 Gitee 账号。之后改完代码只需 `git add . && git commit -m "说明" && git push`。

---

## 第二步：服务器装 1Panel（一次性）

1Panel 是国产开源运维面板，网页点点点管理 Docker、证书、备份，适合没有专职运维的小团队。

### 1. 安装（SSH 登录服务器后执行官方一键脚本）
```bash
curl -sSL https://resource.fit2cloud.com/1panel/package/quick_start.sh -o quick_start.sh && bash quick_start.sh
```
装完它会给出**面板访问地址、端口、用户名、密码**，记下来。

### 2. 开放面板端口
腾讯云控制台 → 安全组 → 放行 1Panel 面板端口（安装时显示的那个）。

### 3. 进面板装 Docker
浏览器打开面板地址登录 → 左侧「容器」→ 若提示未装 Docker，点一键安装。

---

## 第三步：把项目跑起来（一次性）

### 方式 A：用 1Panel 的「应用编排」（推荐，可视化）
1. 面板 →「容器」→「编排」→「创建编排」。
2. 名称 `wxwork`，把本项目 `docker-compose.yml` 的内容粘进去。
3. 在同目录创建 `.env`（参考 `.env.example`），填好：
   - `MYSQL_ROOT_PASSWORD`：强密码
   - `SECRET_KEY`：一长串随机字符串
   - `DEMO_MODE=false`
   - `CORS_ORIGINS=https://wx.zjyxgj.com`
   - `DOUBAO_API_KEY`：豆包密钥（要用 AI 分析就填）
4. 但容器要从 Gitee 拿代码，所以先在服务器拉代码（见方式 B 的 clone 步骤），再用面板编排指向该目录。

### 方式 B：命令行（更直接，复制即用）
SSH 登录服务器：
```bash
# 1. 拉代码
cd /opt
git clone https://gitee.com/你的用户名/wxwork_analytics.git
cd wxwork_analytics

# 2. 配环境变量
cp .env.example .env
vi .env        # 填好密码、SECRET_KEY、DOUBAO_API_KEY 等，DEMO_MODE=false

# 3. 一键起全栈（前端+后端+MySQL）
docker compose up -d --build

# 4. 看状态
docker compose ps
```
起来后，服务监听在 `.env` 里的 `HTTP_PORT`（默认 8080）。

---

## 第四步：绑域名 + HTTPS（一次性）

1. 1Panel →「网站」→「创建网站」→ 反向代理：
   - 域名填 `wx.zjyxgj.com`
   - 代理目标填 `http://127.0.0.1:8080`（上面的 HTTP_PORT）
2. 「网站」→ 对应站点 →「HTTPS」→ 申请 Let's Encrypt 免费证书，一键开启。
3. 腾讯云安全组放行 80 / 443。

完成后浏览器访问 `https://wx.zjyxgj.com` 即生产环境。

---

## 第五步：接入真实企业微信（一次性）

1. 企业微信管理后台 → 自建应用 → 拿 CorpID / AgentID / Secret。
2. 应用「可信 IP」填**服务器公网 IP**；OAuth 回调域名填 `wx.zjyxgj.com`。
3. 登录系统 →「配置后台」填入并保存、测试连接 → 点「同步数据」拉真实客户群。

---

## 日常更新上线（以后每次都只要这样）

**本地：**
```powershell
git add . ; git commit -m "改了xxx" ; git push
```
**服务器：**
```bash
cd /opt/wxwork_analytics
git pull
docker compose up -d --build
```
就这两步。数据库数据保存在 docker 卷 `db_data` 里，更新代码不会丢数据。

---

## 备份（1Panel 自动）
面板 →「数据库」或「计划任务」→ 设置 MySQL 定时备份到本地/对象存储，省心防丢。

---

## 本地开发（改代码时）
双击项目根目录的 `dev.bat`，自动起前后端并打开浏览器（用 SQLite + 演示数据，不碰生产）。
